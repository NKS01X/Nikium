# Nikium

Nikium is a lightweight experimental programming language written in Go.
It uses the `.nik` file extension and is currently under development.

## Features

* Custom syntax designed for simplicity and clarity
* Written in Go for performance and easy maintenance
* Native support for `.nik` source files
* Planned interactive REPL for quick experimentation
* Modular architecture for easy extensibility
* Basic arithmetic and variable handling
* Clear error reporting and debugging support (in progress)
* Designed to be beginner-friendly and educational

## Goals

Nikium aims to provide a simple yet expressive language that is easy to learn and extend.
It's perfect for experimentation with language design concepts and small scripting tasks.
